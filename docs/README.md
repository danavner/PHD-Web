# FastPHD2

FastAPI implementation of web monitor. 

Listens to ```localhost``` on port ```4400``` which is the default port for PHD2. 

## Requirements
Python 3.6+

## Installation

```bash
pip install -r requirements.txt
uvicorn app:app
```

Will run using ```gunicorn``` as a service on 90prime machine.
